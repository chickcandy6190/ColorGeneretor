# ColorGenerator

選択したベースカラーから、相性のいい補色（反対色）を自動計算し、プレビューへリアルタイムに色を反映するカラージェネレーターツールです。

## 開発背景と目的
Web制作やデザインにおいて、ベースカラーに対するアクセントカラー（補色）選びや、実際の画面に適用した際の視認性の確認には手間がかかります。
本アプリは、「色を選ぶと即座にプレビューで見え方を確認できる」ことで、色選びをを効率化することを目的に制作しました。

## 主な機能
- 補色計算: カラーピッカーで色を選ぶと、JavaScriptが255の反転値を計算し補色を生成します。
- プレビュー: 生成された2色をプレビューに自動適用し、具体的な使用感を確認できます。
- 文字色自動調整: 背景色（ベース/補色）の輝度を算出し、文字色を自動で「白」または「黒」に切り替えます。

## 使用技術
- HTML
- CSS
- JavaScript
